BetSocial

BetSocial is a social betting application created for friends and other acquaintances to keep track of various bets made with each other. Once a user has created an account, he or she will be able to connect with other users and create bets. Another user will be set as a mediator to maintain integrity of wagers made. The user pages keep records of all bets created in both the past and the present and the number of bets that have been won or lost. Feel free to use the application by creating a user name or using my log in credentials (username: timlee, password: 123456). The languages and frameworks used include React, Passport.js, MaterialUI, Bootstrap, and Mongoose.

Diary

This application provides a great medium for travelers to document their trips from local areas to countries abroad. Users of Diary are able to visually log their trips on google map for other users to see! With each location logged on the application, users are able to upload photos and provide a brief blog post about that particular logged location. Database currently in work. Main languages and frameworks include Moment.js, Firebase, Bootstrap, and Google Maps API.
